This is a non-custodial app: Olynt Pay never holds customer funds. Payments go wallet-to-wallet on-chain.

How it works

  1. Merchant opens POS and enters amount (or creates a payment link).
  2. Olynt Pay generates a QR code + shareable link.
  3. Customer scans, connects a wallet, and pays.
  4. Funds go directly to the merchant wallet on-chain.
  5. Merchant verifies using the transaction hash (receipt).
Recommended: customers should pay using a wallet browser for smoother approvals (Coinbase Wallet / MetaMask).
